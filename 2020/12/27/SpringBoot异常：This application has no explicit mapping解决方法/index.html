Spring-boot异常：This application has no explicit mapping解决方法

2020-12-27

运行springboot的时候报错，This application has no explicit mapping其他配置都没有问题。

一、错误提示：


Whitelabel Error Page

This application has no explicit mapping for /error, so you are seeing this as a fallback.

Mon Jun 24 14:56:23 CST 2019

There was an unexpected error (type=Not Found, status=404).

No message available

二、原因：

目录的问题，Application启动类的位置不对，Application一定要包含所有子包，即只要把Application放在最外层的包里就没有问题了，我的GreetingController放在了外层的包里面，导致找不到页面。

三、解决方法：

这里有两种解决方法：

解决方法1:移动控制层的包：

很简单，就是把Application放在最外层的包里就行了。

成功运行。

解决方法2:添加注释：

在启动的Application的类中，添加注释，指定你的GreetingController位置，就可以指定加载，解决问题。@SpringBootApplication(scanBasePackages=”controller”)

四、汇总可能会出现报错的原因：

出现这个异常说明了跳转页面的url无对应的值。
